# FXRQuery
A tool for gathering data about FXRs.
### Supported Games
- Dark Souls 3
- Elden Ring
## Usage
1) Put the full path to the game directory in Config.xml.
2) Download a .csv file from an FXR spreadsheet and drop it on the executable.
